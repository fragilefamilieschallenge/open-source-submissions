This file describes the python files in this project:

1. HW2_GPA : 
This python script implements the regression methods for estimating the 
GPA performance. It includes Lasso, ElasticNet and SVR along with
their evaluation on various parameters such as MSE, R^2 and
explained variance score.

2. feature_extract : 
This python script analyzes the features extracted by the Lasso regression based on their coefficient values. The top 10 coefficients are selected.
